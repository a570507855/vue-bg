# 目录结构：
    components/         框架模板组件
    css/                通用css样式    
    icons/              通用图标
    img/                图片资源
    router/             页面路由
    script/
          ├─ filter/    vue过滤器
          ├─ lib/       通用工具
          ├─ uc/        通用vue组件
    store/              vue全局状态管理
    views/              页面
    main.js             入口js
    permission.js       vue-router全局前置守卫
    setting.js          基础配置数据

# 安装&运行调试
    1. npm i
    2. npm run build(打包)
    3. npm run dev
